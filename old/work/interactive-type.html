INTERACTIVE TYPE: 空

2021

Creating an interactive poster, exploring how typography and the visuals of a word can inform.
Uses p5.js textToPoints() function to get the outline of the letterform.

空(kong)(Chinese): empty, hollow, void, vacant, unoccupied

→ link to code

Ideation & Iteration

The three words I initially decided on were "空"(empty), "shy", and "bloom". More than anything, I wanted to encorporate Chinese into this project, after missing the chance to with Itchy from the previous semester. Thinking about how I could use textToPoints() to transform I came up with "bloom", and I tried to be more obvious in encorporating my own personality through choosing "shy".
After iterating on what I could do with these three words, it became clear that my ideas for "空" and for "shy" were a bit too similar, so I needed to change one. The third iteration of "shy" with the fracturing and cursor-avoidance inspired a stronger tone and led me to change the word to "NO". In my prototypes I replaced the Chinese character with the letter 'm' as I had yet to find a Chinese OTF font of a small enough file size to upload.

Reflection

This project was really fun. It was interesting to think about and use the camera in an alternate method, not to necessarily capture a photo or image but building a tool that can take parts of that infomrmation to create something new. I basically made a camera-based drawing software, and I really like how "undetermined" the result of using this tool is. Even though I'm the one determining the ranges for the colors, the user has to use things from their own environment to be able to draw and it's what the user is presenting that is represented on the canvas. So it's a collaborative tool in a way, that requires both the developer and the user to contribute for it to work.
