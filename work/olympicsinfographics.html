moodboard & concept

    The goal was to first create an almost complete set of graphics that would act like a "template" so I'd be able to easily & quickly input new data, simulating a live coverage situation.

    I started with three moodboards to work out a visual style I liked. I ended up combining the first and last for a scientific feel with tactile/scrapbook-y visuals not traditionally seen in data viz. I wanted to make my infographics feel fun and approachable but also informative and accurate.

    Using data from the previous Winter Olympics (Pyeongchang 2018), I started on my design. I wanted all of my panels to exist within the same design family, but let each sport have its own flavor within that as well. That meant using the same typeface, image treatments, and organization system across the four sports, but giving each their own color and dynamic pattern.
